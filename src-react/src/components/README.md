# Components Notes

本目录里与 AI 和修订相关的核心组件，已经从早期“DiffView / AI Chat”形态逐步演进为面向小说写作的工作台组件。

## Key UI Areas

### `AIChatPanel`

当前已经是一个多工作区侧栏，而不是单一聊天框：

- `Chat`
- `Planner`
- `Review`
- `Memory`

它负责承载：

- 会话管理
- 当前文档上下文
- 模型恢复入口
- 规划任务操作
- 修订建议审阅
- 长期记忆查看与治理

### `DiffView`

虽然组件名仍保留 `DiffView`，但产品语义已经更接近“修订建议审阅器”：

- 查看原文与建议稿
- 接受 / 忽略单条建议
- 接受 / 忽略整组建议

### `NovelStructurePanel`

承担作品导航，而不是通用文件树：

- 章节
- 大纲
- 设定
- 计划

### `HistoryPanel`

承担版本记录，而不是工程历史面板：

- 快照浏览
- 版本恢复
- 当前文档过滤

## Current Product Direction

组件设计应该优先支持：

1. 长篇写作流程
2. AI 协作与审阅
3. 可见、可控的长期记忆
4. 作者对结果的理解与掌控

如果某个组件改动更像“编程 IDE 功能”，应谨慎评估是否真的适合当前产品方向。
