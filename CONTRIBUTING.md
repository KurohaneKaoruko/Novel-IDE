# Contributing to Novel-IDE

感谢你对 Novel-IDE 的兴趣！欢迎贡献代码。

## 开发环境

1. Node.js 20+
2. npm 9+

```bash
npm install
npm run dev
```

## 代码规范

- 使用 ESLint + Prettier
- TypeScript 开启 strict 模式
- 提交前运行 `npm run lint`

## 提交规范

使用 Conventional Commits：

- `feat:` 新功能
- `fix:` Bug 修复
- `docs:` 文档更新
- `style:` 代码格式
- `refactor:` 重构
- `test:` 测试
- `chore:` 构建/工具

## Pull Request

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/xxx`)
3. 提交更改
4. 推送到 Fork 的仓库
5. 创建 Pull Request

## 问题反馈

使用 GitHub Issues 报告 Bug 或功能请求。
